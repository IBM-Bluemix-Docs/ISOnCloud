# Getting started with IBM Information Server on Cloud
{: #iisoc_getting_started}

IBMÂ® Information Server on Cloud offerings provide data integration and governance products as a service on the global cloud infrastructure of IBM SoftLayer. It offers the rich features of an on-premises data integration product deployment without the cost and complexity of deploying the infrastructure. Optionally managed add-on services can also be added to maintain and manage the infrastructure.
{: shortdesc}
